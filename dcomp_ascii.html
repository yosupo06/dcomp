dcomp.ascii

  • ASCIIString

    Declaration

    struct ASCIIString;

    ASCII文字限定string

    Discussion

    中身をASCII文字に限定したrange系の操作を実装したstring.

    速度に困らないなら普通にstringを使ったほうが良いと思う.

    Examples

    1. import std.algorithm; import std.range.primitives; auto s = "タネなし手品"; auto asc = s.ASCIIString; assert(s.front == 'タ'); assert(asc.front == "タ"[0]); assert(s.back == '品'); assert(asc.back == "品"[$-1]);
