  • fixed

    Declaration

    T[N] fixed(T, size_t N)(T[N] a);

    静的配列のリテラルであると明示的に指定する

    Examples

    1. auto a = [[1, 2].fixed]; assert(is(typeof(a) == int[2][]));

  • FastAppender

    Declaration

    struct FastAppender(A);

    std.appenderをより高速化したもの.

    Discussion

    参照型のように動くが, コピーした後に操作するとそれ以外の参照が全部壊れる. ブロック外に出さない一時バッファのような使い方を想定している. 速度が重要でないならばstd.appenderを使うこと.
