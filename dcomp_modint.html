  • DModInt

    Declaration

    struct DModInt(string name);

    自動mod取り構造体(実行時mod指定)

    Examples

    1. alias Mint1 = DModInt!"mod1"; alias Mint2 = DModInt!"mod2"; Mint1.MD = 7; Mint2.MD = 9; assert((Mint1(5)+Mint1(5)).v == 3); // (5+5) % 7 assert((Mint2(5)+Mint2(5)).v == 1); // (5+5) % 9
